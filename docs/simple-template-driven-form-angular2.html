Before some days ago, Angular2 released its RC5 version & I have updated all my existing apps from RC4 to RC5 version. This tutorial is written using RC5 version & I will keep on updating this tutorial with upcoming versions too. In this tutorial I will show you how to create a simple form in Angular2, actually there are two ways by which you can make forms in AngularJS. One way is template driven that I will show in this tutorial & other way is by using model driven form that I will show you in upcoming post. If you are learning angular2 then I would say first learn template driven forms then go for model driven. Enough talking, let’s get started now.

Steps to be completed before learning this:

  • First create a Hello World app – use this official guide to complete this step.
  • Choose typescript while building Hello World app from Angular2 in above tutorial. Don’t worry if you are not familiar with typescript, it is like a super set of JavaScript with some added functionality.
  • Make some changes in your app.module.ts file if you want to use forms in Angular2 RC5 version. Read this tutorial for required changes
  • That’s all, we are ready to create forms now.

As you can there is nothing special in the code above, just a pure html form syntax. Now its time to add some Angular2 controls. I am assuming you know about data bindings in Angular2 if not then wait for my next tutorial, I will explain you.
Here in above form, we have 2 inputs, so we will add two way data binding in our two inputs using “banana in square” syntax like this [(ngModel)]=”logindata.email” & [(ngModel)]=”logindata.password” so we can communicate out data from component as well as view side.
After adding data binding, we will add form submit function syntax in our view like this  (submit)=”performLogin(logindata)”
Let’s add all these things in our form & check out our updated code. It will look like this :

export class UserLoginComponent{
logindata: any ={};
constructor(){
};
performLogin(logindata){
console.log(logindata)
// perform all actions here with you logindata now
};
}
Let’s try to understand our component now, logindata is our object that will take data from html form using name attribute & two-way data binding. After clicking on form submit our performLogin() function will be evoked & in this function we can perform you after form submit task like sending login data to any api etc.

Using Validation in template driven form:

As you do validation in angularJS v1.x , here also you can use these three form fields to check its state , change, error etc & add validation message or can change color etc in your form.
  • touched or untouched
  • valid or invalid
  • pristine or dirty
As you can easily see template driven forms are similar to AngularJS v1.x & hence if you are familiar with Angular version 1 forms then you can easily create simple form in Angular2 also. I personally love model driven forms more than template driven as they provide many new libraries & make our validation task easier.

Conclusion:

Now you have learned creating simple form in Angular2 using template driven method, I would advice you to try to learn model driven forms also as this tutorial was just to give a basic understanding of forms in Angular2.
