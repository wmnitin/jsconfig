In this tutorial I am going to introduce you some effective ways of using AngularJS Controllers, As I said in my previous article of AngularJs first app configuration  that except this first app, you will be using AngularJS controllers to create your app. AngularJS Controllers controls the flow of data in your application , from fetching of data from API to saving your HTML form data to API.  Our AngularJS version 1.x is similar to MVC pattern (Model-View-Controller) & controller part is everything which works between your model data & html views. So, now lets move forward to defining controllers.

Defining AngularJS Controllers:

If you remembers ng-app=”app-name” is the first directive in any AngularJS app, so first of all we will initialize our app using by creating module of app-name.
1
var JsConfigApp = angular.module('app-name', [])     //intialization of JsConfigApp
Now lets define any controller now, lets take a name “firstCtrl”
1
2
3
JsConfigApp.controller('firstCtrl', function($scope){
   $scope.customtext='I am inside firstCtrl'
})
we are done with our first simple controller “firstCtrl” , Now you must be wondering about $scope variable. $scope is a variable that holds your data inside controller, once you switch to another controller then you can not use scope data of another controller, for making this work we use $rootScope, Read this post $scope vs $rootScope difference in AngularJS for better understanding between $scope & $rootScope.
In above controller, I am assigning the data value to customtext , now you can fetch this value from your HTML using ng-bing directive or {{ }} expression. Let me show you.

   JsConfigApp.controller('firstCtrl', function($scope){
      $scope.customtext='I am inside firstCtrl'
   })
</script>
</head>
<body ng-app="<strong>app-name</strong>">
<div ng-controller="<strong>firstCtrl</strong>">
   Where are you ? Answer : {{<strong>customtext</strong>}}
</div>
</body>
</html>
Make sure you include AngularJS’s core JavaScript file above your controller’s JavaScript file in head tag of your HTML page.
Working: Whenever this html file will render by any browser, it will search for firstCtrl controller in your JavaScript file, After that it will search for customtext whether it is defined any value in controller or not. After finding its value in controller it will display us the defined value like this.
1
<strong>Output:</strong> Where are you ? Answer : I am inside firstCtrl

Now our task is to show both “name” & “email” below form after clicking “show data” button. Lets create our controller “myCtrl” , assuming you are saving this code in external JavaScript file name “app.js”
1

var JsConfigApp = angular.module('app-name', [])
JsConfigApp.controller('myCtrl', function($scope){
   $scope.showData=function(){
      $scope.finalData = $scope.myname + " " + $scope.myemail;
   }
})
Here I have defined method name “showData” inside controller, analyse this code & now you have learned using variables & methods inside AngularJs controllers.
That’s it. Now try running this html in your browser & click show data after filling both fields of form & you will see your desired result just below your button. You can also create custom directives in AngularJS, read more : Coding with AngularJS custom directives
